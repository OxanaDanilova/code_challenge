## How to install

### Backend part

```
npm i
create file .env
npm start
```

### Frontend part

```
npm i
npm start
```

---

## API Endpoints:

### GET /users

The list of the users with names and surnames

### POST /users

Create new user

### GET /users/:id

Find the user with messages and subscriptions

### POST /messages

Create the message
